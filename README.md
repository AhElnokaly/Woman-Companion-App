# 🌸 Woman Companion (رفيق المرأة)

تطبيق ذكي، متكامل، وأوفلاين بالكامل مصمم ليكون الرفيق المثالي لكل امرأة في مراحل دورتها الشهرية، التخطيط للإنجاب، الحمل والنفاس، مع دمج مميز للصحة النفسية، الجسدية، العبادات، والترطيب اليومي بخصوصية مطلقة وأمان تام.

---

## ✨ المميزات الرئيسية للمشروع (Core Features)

### 1. 🧭 واجهة التنقل بالإيماءات والسحب (Swipe Navigation)
* **سلاسة فائقة**: تنقل كامل بين صفحات التطبيق الخمس (الحمل، الدورة، الغذاء والماء، الأعراض، الأدوات) عبر السحب لليمين واليسار (`HorizontalPager`) مع انتقال بصري ناعم ومريح.
* **مزامنة تامة**: شريط التنقل السفلي متزامن بالمليمتر مع حركة السحب لعرض التبويب النشط فورياً.

### 2. 🪟 واجهة التصميم البلورية (Glassmorphic Design)
* **جمالية عصرية ومريحة للعين**: واجهة مستوحاة من نمط الزجاج البلوري (`Glassmorphic Cards`) الذي يعزل العناصر بخلفية غائمة أنيقة وحدود متوهجة خفيفة لتوفير مظهر متميز وأنيق.
* **تفاعل حي**: أزرار بلورية متحركة (`AnimatedGlassButton`) تستجيب بالضغط والارتداد الفيزيائي اللطيف وتضفي حيوية لتجربة الاستخدام اليومي.

### 3. 🌤️ شريط طقس جوجل الفوري مع التوجيهات اليومية (Google Weather Header)
* **تحديث فوري للطقس**: يعرض درجات الحرارة والنسبة المئوية للرطوبة في أعلى شاشات التطبيق كأول عنصر واجهة يراه المستخدم.
* **أيقونات طقس تفاعلية**: شمس ساطعة متوهجة ☀️، سحب غائمة متحركة ☁️، أو أمطار باردة 🌧️ بناءً على درجات الحرارة وسرعة الرطوبة.
* **توصيات جوري للطقس**: يقدم التطبيق نصائح فورية دقيقة مرتبطة بالجو (مثل زيادة أهداف الترطيب وشرب المياه في الأيام الحارة، أو شرب الينسون الدافئ في الأيام الممطرة والباردة).

### 4. 🗄️ قاعدة بيانات الأكلات والمشروبات المصرية (Egyptian Food Database & NLP Log)
* **قاعدة بيانات محلية غنية**: تضم الأكلات والمشروبات المصرية التقليدية واليومية (مثل: ملوخية بالدجاج، أرز بالبشاميل، شاي، قهوة باللبن، كشري، فول، فول بالزيت، بيض مسلوق، وغيرها الكثير).
* **معالج لغة طبيعية محلي (NLP Parser)**: يمكنكِ أن تكتبي لجوري ببساطة: *"شربت قهوة باللبن"* أو *"أكلت طبق ملوخية وأرز وقطعة دجاج"* وسيقوم محرك الذكاء الأوفلاين فورياً بتفكيك الجملة، حساب السعرات الحرارية، البروتينات، والكربوهيدرات بدقة وإضافتها لسجل تغذيتكِ!
* **الإكمال التلقائي والاقتراحات (Autocomplete & Suggestions)**: يعرض حقل التغذية اقتراحات ذكية خافتة متوقعة أثناء الكتابة لتسهيل الإدخال السريع.

### 5. 🤖 صديقتكِ الذكية جوري (Jouri Matrix - 500+ Interaction)
* **أكثر من 500 نصيحة وتفاعل ديناميكي**: نظام مصفوفة ذكي ومتطور يمنح جوري ردوداً وتفاعلات متغيرة باستمرار حسب:
  * حالتكِ المزاجية (حزينة، سعيدة، قلقة، مرهقة).
  * الطقس الفعلي ومستوى نشاطكِ.
  * أطوار دورتكِ الشهرية وحالة حملكِ الحالية.
  * التذكير المستمر بأخذ المكملات الغذائية، الفيتامينات، وحمض الفوليك.
* **توجيه سريع ومباشر**: عندما تكتشف جوري وقت فراغ أو إهمال للترطيب، تقترح عليكِ تجربة ميزة معينة في التطبيق مع توفير زر انتقال فوري وسلس لتلك الصفحة بلمسة واحدة.

### 6. 🌙 المظهر الداكن والفاتح (Dynamic Light & Dark Themes)
* **تحكم مطلق**: دعم كامل للمظهر الداكن الفاتن والراقي والوضع الفاتح المريح المشرق، يمكن التبديل بينهما فورياً بلمسة واحدة من صفحة الإعدادات لتتوافق الواجهة والخطوط والظلال بسلاسة فائقة.

### 7. 🐾 عداد الخطوات الذكي بالخلفية (Dynamic Background Step Counter)
* **قراءة مباشرة من المستشعرات**: يدعم التطبيق القراءة الفورية والدقيقة من مستشعرات الهاتف الرياضية المتخصصة (`Sensor.TYPE_STEP_COUNTER` و `Sensor.TYPE_STEP_DETECTOR` مع توفير بديل متطور بمستشعر التسارع `Sensor.TYPE_ACCELEROMETER` للهواتف القديمة).
* **يعمل حتى لو كان التطبيق مغلقاً (Background & Boot-Resistant)**: تم بناء الخدمة باستخدام خدمة أمامية مستمرة (`Foreground Service`) من فئة المراقبة الصحية والنشاط البدني (`health`) لضمان عدم توقف احتساب خطواتكِ حتى بعد إغلاق التطبيق نهائياً أو إعادة تشغيل الهاتف بفضل مُستقبِل بدء التشغيل التلقائي (`BootReceiver`).
* **إشعار حي دائم ومبهر**: يعرض إشعار النظام المخصص خطواتكِ الحالية المقطوعة فوراً مع الحساب والمزامنة اللحظية بداخل قاعدة بيانات `Room` المحلية، ليضمن عدم ضياع أي خطوة أو احتسابها مرتين بفضل خوارزمية ذكية تحتفظ بالخطوات التراكمية بأمان في `SharedPreferences`.

### 8. 🌙 محلل ومراقب النوم الذكي (Smart Sleep Analyzer)
* **تتبع شامل**: احتساب فترات النوم بالملي ثانية وتصنيف جودة النوم (نسبة مئوية) مع تحليل توزيع النوم عميق/خفيف/حركة سريعة (REM).
* **نصائح مخصصة**: تزويد المستخدمة بإرشادات طبية للنوم الآمن أثناء الحمل (مثل النوم على الجانب الأيسر) وتأثير المشروبات والمنبهات.

### 9. 🔗 رابط الرفيق ومشاركة الشريك (Companion Sync Link)
* **مزامنة محلية مشفرة ومستقلة**: رمز ربط فريد لمشاركة حالتك الصحية والعبادات وأهداف المشي والمياه مع زوجك أو عائلتك دون رفع بياناتك إلى خوادم وسيطة عامة.
* **تنبيهات ورسائل دعم فورية**: تمكين الشريك من محاكاة وإرسال رسائل تشجيع مخصصة وتذكيرات بالترطيب أو متابعة حركة الجنين تظهر فورياً في لوحة الإشعارات المزدوجة بذكاء تام.

### 10. 💊 الصيدلية المنزلية المتقدمة وخزانة الأدوية (Advanced Home Pharmacy Cabinet)
* **مراقبة المخزون الفعلي**: تتبع عدد الحبوب أو الجرعات المتبقية في العبوة بدقة مع تنبيه ذكي متقدم للمخزون الحرج عندما يقل المتبقي عن 5 جرعات.
* **مراقبة تواريخ الصلاحية والأمان**: كشف تواريخ الصلاحية المنتهية أو القريبة (أقل من شهر) وعرض تصنيف أمان استخدام الدواء أثناء فترة الحمل والرضاعة (آمن، بحذر، استشر دكتور).
* **زر الجرعة التفاعلي**: زر تفاعلي بنقرة واحدة لتسجيل تناول جرعة الدواء فورا وتحديث المخزون المتبقي تلقائياً في قاعدة البيانات.

### 11. 🔗 مزامنة مصفوفة جوري عبر GitHub وتخصيص الـ API
* **تحديث مستمر للذكاء**: يمكنكِ تخصيص ورسم مصفوفتكِ الخاصة وتحديثها ديناميكياً بوضع رابط ملف الـ JSON الخاص بكِ على GitHub لتنزيل النصائح الجديدة فورياً وتحديث ذكاء جوري دون الحاجة لتحديث التطبيق بأكمله!
* **مفتاح API خاص**: يتيح لكِ التطبيق تزويد جوري بمفتاح الـ Gemini API الخاص بكِ لتشغيل الذكاء الاصطناعي السحابي المتطور عند الرغبة، مع احتفاظ التطبيق بالوضع الذكي الأوفلاين كخيار أساسي دائم.

## 🏗️ التقنيات المستخدمة (Tech Stack)

* **لغة البرمجة**: Kotlin بالكامل.
* **بناء الواجهات**: Jetpack Compose مع Material Design 3.
* **التنقل الحركي**: Compose Foundation Pager (`HorizontalPager`).
* **حفظ البيانات**: Room Database لتخزين السجلات الصحية والتغذية والخصوصية بشكل آمن ومشفر بالكامل على جهازك.
* **تحديث البيانات وسحب الملفات**: OkHttp للحصول على تحديثات مصفوفات جوري من GitHub وجلب الطقس محلياً.

## 🔒 الخصوصية والأمان الفائق (Stealth Mode)

تطبيق **رفيق المرأة** أوفلاين بالكامل، لا يتم إرسال أي من بياناتك الصحية أو الدورة أو وجباتك إلى خوادم خارجية إلا عند مزامنة جيت هاب الاختيارية أو استخدام الـ API. يدعم التطبيق رمز قفل PIN وتفعيل وضع التخفي لإخفاء صور وتفاصيل الحمل من المتطفلين في شاشة القفل.

🌸 **صُنع بحب ودقة ليكون رفيقك المخلص والأكثر ذكاءً وأماناً.**
